# 📚 Book Bank Management System

## Description

The Book Bank Management System is a simple web application developed to help schools or colleges manage books and student borrowing records. It replaces manual record keeping with a computerized system that is easier to use and manage.

## Features

- User login
- Add, edit and delete books
- Register students
- Issue books
- Return books
- Calculate overdue fines
- Search books and students
- Generate basic reports

## User Roles

### Administrator
- Manage books
- Manage user accounts
- View reports
- Manage fine payments

### Librarian
- Register students
- Issue books
- Return books
- Search records

### Student
- View available books
- View borrowed books

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Installation

1. Install XAMPP.
2. Copy the project folder into the **htdocs** folder.
3. Start Apache and MySQL.
4. Create a MySQL database using phpMyAdmin.
5. Import the SQL database file.
6. Update the database connection in `db.php`.
7. Open your browser and visit:

```
http://localhost/BookBankManagementSystem/
```

## Project Purpose

The aim of this project is to make it easier to manage books, students, borrowing, returns, and overdue fines while reducing paperwork and improving record keeping.
